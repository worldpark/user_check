  - 01: 로그인 및 페이지 에러

    - 02: 비활성화 계정
    - 03: 만료된 계정

    - 07: 인증할 수 없는 정보

    - 03: 인증 정보 잘못됨
    - 04: 인증 정보 올바르지 않음

    - 03: 갱신 정보가 올바르지 않음
    - 04: 갱신 정보가 잘못됨
